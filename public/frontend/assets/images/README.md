# صور الموقع

هذا المجلد يحتوي على صور الموقع. يمكن استبدال ملفات SVG المؤقتة الحالية بصور PNG/JPG فعلية عند توفرها:

- **favicon.svg** — أيقونة الموقع (تبويب المتصفح). يمكن استبدالها بـ `favicon.ico` أو PNG وتحديث مراجعها في الـ `<head>`.
- **logo.svg** → استبدل بـ logo.png (الشعار، مقترح 45×45 بكسل)
- **trainer.svg** → استبدل بـ trainer.png (صورة المدرب، مقترح 350×350 أو أكبر)
- **course-webdev.svg**, **course-python.svg**, **course-mobile.svg** → غلافات الكورسات (مقترح 400×200 أو 16:9)
- **workshop.svg** → صور معرض النشاطات (مقترح 400×250 أو أكبر)

بعد إضافة الصور الفعلية، حدّث مراجع الملفات في HTML من `.svg` إلى `.png` (أو الامتداد المستخدم).
